# Need For Speed
วิธีเล่น
  Player1 ควบคุมโดยใช้ WASD ,Player2 ควบคุมใช้ลูกศรบนKeyboard หรือ Arrow โดยเกมจะให้บังคับBigbikeของเราให้หลบ 
  Object รถต่างๆที่อยู่บนถนน เรื่อยๆจนกว่าจะมีPlayerคนใดคนนึงตาย และPlayerที่รอด ให้เล่นไปเรื่อยๆจนกว่าจะตาย โดยจะมี
  Record มีScore ให้ดูว่าได้คะแนนเท่าไหร่ และจะโชว์ของคนเยอะสุดตอนจบเกมและแสดงชื่อคนชนะตอนจบเกมเช่นกัน
